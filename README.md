# auto-clicker
Allows doing automatic clicks by mouse. E.g. for some kind of games where you need periodically click on buttons to get reward.

## Technology stack
**Kotlin, Swing, Gradle** 

## How to run?
Build fat jar
> ./gradlew shadowJar

Run application
> java -jar build/libs/auto-clicker-all.jar
